# chip-8-disassembler
Disassembler for the CHIP-8 bytecode

## Build Instructions

Dependencies are CMake and Doxygen

### To build

In project root run

1. mkdir build
2. cd build
3. cmake ..
4. make
5. ./chip-8

### To clean

either `rm -R build` or `git clean -fdx`

### To generate docs

In project root run `doxygen`
